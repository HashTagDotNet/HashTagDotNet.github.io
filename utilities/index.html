Collection of reusable utilities and functions. Used by most projects in this suite.

Collections - Handles things like data structures and groups of objects

Configuration - Retrieve strongly typed values from .config and .config token replacement

Cryptography - Encode & decode objects using internal and external keys

Diagnostics - A log wrapper to easily integrate logging and tracing

i18N - Utilities around dealing with various kinds of spatial, globalization, and internationalization

DistanceSpan - operations and conversions around distance. Supports both Metric and English systems.

IO - Deals with streams and file based operations

Net - Operations relating to network protocols: MIME helpers

Reflection - Interrogates runtime environment

Serialization - Lightweight wrappers around common conversion patterns

ServiceModel - Patterns around WCF

Text - String construction and manipulation

Web - Web centric helper methods

(core) - Functionality that doesn't fall within other namespaces.

Validation - Methods to validate data, particularly used in parameter validation areas

Transform - Modeled after .Net Convert methods

Utils.Date - A DateTime library

Extensions - Wrappers around existing library methods.

CoreConfig - All configuration that is specific to the HashTag.Core classes
